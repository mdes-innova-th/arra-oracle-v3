# Constitution: #2675 taskmap pilot

- Workers must ACK with `maw hey` before touching files.
- A `[P]` task is honest only when its listed files do not overlap another task in the same wave.
- Shared generated files, version bumps, and changelog edits stay sequential unless a lead splits them further.
- Done means scoped tests pass, touched files are reported, and the worker marks its task complete.
